Title: MXA310 Crestron/AMX Control Strings
Id: 4871
Legacy Id: 6059

Question:
I want to control the MXA310 with a Crestron touchscreen. What are the API control strings?


Answer:

	The MXA310 can be controlled by AMX, Crestron, Extron, etc.
	
	The touch screen can control/monitor:
	
		Volume Controls 
	
		Mute
	
		Metering
	
		Channel and Device Names
	
		LED Controls
	
		Gate Indicators
	
		More ...

	The control strings can be found in the MXA310 User Guide.
